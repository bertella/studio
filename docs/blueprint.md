# **App Name**: DigitalBridge Consulting

## Core Features:

- Hero Section Display: Showcase a compelling 'Digital Transformation for Small Businesses' header with a strong call to action or mission statement.
- Services Overview: Clearly list specialized services including Inventory Systems, Automated Budgeting, and SQL Optimization.
- Interactive Portfolio Grid: Display a grid of portfolio projects, prominently featuring the hardware store inventory app, with navigable details for each entry.
- Technical Skills Showcase: Dedicated section detailing key technical expertise, specifically Relational Databases and AppSheet proficiency.
- Project Description Generator (AI Tool): An internal, AI-powered tool to assist the consultant in drafting compelling and tailored project descriptions for new portfolio entries based on project type and outcomes, ensuring alignment with client needs and target audiences.
- Client Contact Form: A straightforward form enabling potential clients to easily get in touch for inquiries and collaboration opportunities.

## Style Guidelines:

- Dark Theme Background: A very dark, almost black, background with a subtle blue tint for a professional, deep, and modern aesthetic. RGB: #0E0F11
- Primary Color: A sophisticated, rich indigo blue used for key elements, headings, and branding that signals trust, expertise, and digital innovation. RGB: #233D75
- Accent Color: A vibrant yet clean light cyan/aqua used for calls to action, interactive elements, and important highlights, providing modern contrast and drawing attention. RGB: #74D4F7
- Primary Font: 'Inter' (sans-serif) for all text elements. This choice provides excellent readability, a modern and objective feel, and ensures clean aesthetics for both headlines and body copy.
- Employ minimalist, outline-style icons with a modern geometric quality to maintain a clean, corporate visual language that complements the dark theme. Ensure icons are optimized for scalability and clarity across all devices.
- Implement a clean, spacious layout characterized by ample negative space to highlight content, a clear hierarchy of sections for intuitive navigation, and full responsiveness across diverse screen sizes.
- Incorporate subtle, professional micro-interactions for hover states, button clicks, and section transitions. Animations should enhance user engagement without causing distraction or reducing performance, aligning with a high-end corporate feel.
